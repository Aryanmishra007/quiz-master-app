### Setup Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/Aryanmishra007/quiz-master-app.git
   ```
2. Navigate to the project directory:
   ```bash
   cd quiz-master-app
   ```
3. Install the dependencies:
   ```bash
   npm install
   ```
